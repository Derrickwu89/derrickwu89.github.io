Enumerator

11/30/2014

There are many enumerable methods that can be use in Ruby programming. There are few that can be consider very useful in most circumstance. Such as #map, #group_by, and # cycle

Today, I will write a little example that explain one of those three. I will talk about the #map. Enumerable#map is very useful when we can to output a complete new array in changing every element of the old array with some sort of method that we input.

This is an example of the Enumerable#map

arr =[1, 2, 3, 4, 5]

new_arr = arr.map{|x| x*2}

This will generate a new array call new_arr and the element inside of the old arr arry will transfor and each of them will will follow the method that we set up for x. Each element [1, 2, 3, 4, 5] will multiply by 2. Now you will have a new array call new_arr with new elements. The new element will be:

new_arr = [2, 4, 6, 8, 10]

Enumerator#map is invoked by a collection and accepts a block of code that acts upon each element in that collection and return the transformed collection.
