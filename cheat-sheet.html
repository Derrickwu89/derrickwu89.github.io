Ruby Enumerable

Each

Each is an iterator that cycles through each element in an array and performs a given action on each element. 
            ['cat', 'dog', 'pig', 'goat'].each{ |animal| puts animal }
            #=>   cat
            #=>   dog
            #=>   pig
            #=>   goat

            OR:

            ['cat', 'dog', 'pig', 'goat'].each do |animal|
            puts animal
            end

Map

Map will iterate through each element in an array and perform a given action on each element
and return a new transformed array. 
            arr = [1,2,3,4,5]
            brr = arr.map{|x| -x}

            puts arr.join(", ")   #=>   1, 2, 3, 4, 5   
            puts brr.join(", ") #=> -1, -2, -3, -4, -5

Inject

The inject method takes a collection and reduces it to a single value, such as a sum of values. 
  
            val = [1,3,5,7].inject(0) do |total, num|
                total += num
            end   
            puts val   #=> 16

Sort_by

Sort_by will sort an array by a given arguement. 
            ["apple", "pear", "fig"].sort_by {|word| word.length}
            #=> ["fig", "pear", "apple"]

Take_while

Take_while will return items in an array that fit the given arguement. 
            a = [1, 2, 3, 4, 5, 0]
            a.take_while {|i| i < 3} 
            #=> [1, 2]

Find_all

Find_all will scan through an array and return any items that fit the given arguement. 
            (1..10).find_all {|i| i % 3 == 0}
            #=> [3, 6, 9]

Find_index

Find_index will scan through an array and return the location of any item that fits the given arguement. 
            (1..10).find_index {|i| i% 5 == 0  and i % 7 ==0} 
                #=> nil
            (1..100).find_index {|i| i% 5 == 0  and i % 7 ==0} 
                #=> 35
            (1..10).find_index(50) 
                #=> 49
